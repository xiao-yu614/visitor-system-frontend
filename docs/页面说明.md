# 页面说明

## 页面列表

### 访客端页面

#### 申请页面
- **路径**: `/`
- **组件**: `src/views/visitor/ApplyPage.vue`
- **说明**: 访客填写访校申请信息的页面

### 认证页面

#### 登录页面
- **路径**: `/login`
- **组件**: `src/views/auth/LoginPage.vue`
- **说明**: 管理员登录页面

### 管理端页面

#### 仪表盘
- **路径**: `/admin`
- **组件**: `src/views/admin/DashboardPage.vue`
- **说明**: 管理后台首页，展示统计数据和图表

#### 申请列表
- **路径**: `/admin/apply-list`
- **组件**: `src/views/admin/ApplyListPage.vue`
- **说明**: 访校申请列表，支持审批操作

#### 来访记录
- **路径**: `/admin/visit-record`
- **组件**: `src/views/admin/VisitRecordPage.vue`
- **说明**: 来访记录管理，支持签到签退

## 布局组件

### AdminLayout
- **组件**: `src/components/layout/AdminLayout.vue`
- **说明**: 管理后台布局容器

### HeaderBar
- **组件**: `src/components/layout/HeaderBar.vue`
- **说明**: 顶部导航栏

### SideMenu
- **组件**: `src/components/layout/SideMenu.vue`
- **说明**: 侧边导航菜单

## 路由结构

```
/                    # 访客申请页
/login               # 登录页
/admin               # 仪表盘
/admin/apply-list    # 申请列表
/admin/visit-record  # 来访记录
```

## 页面权限

| 页面 | 是否需要登录 | 角色要求 |
|------|-------------|---------|
| `/` | 否 | 无 |
| `/login` | 否 | 无 |
| `/admin` | 是 | 管理员 |
| `/admin/apply-list` | 是 | 管理员 |
| `/admin/visit-record` | 是 | 管理员 |
