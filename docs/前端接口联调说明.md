# 前端接口联调说明

## 接口基础信息

- 接口地址：`http://localhost:8080/api`
- 认证方式：JWT Bearer Token
- 内容类型：application/json

## 接口列表

### 认证接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 登录 | POST | /auth/login | 用户登录 |
| 登出 | POST | /auth/logout | 用户登出 |
| 获取当前用户 | GET | /auth/user | 获取当前用户信息 |

### 访客申请接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建申请 | POST | /visit-apply | 创建访校申请 |
| 获取申请详情 | GET | /visit-apply/{id} | 获取申请详情 |
| 更新申请 | PUT | /visit-apply/{id} | 更新申请信息 |
| 删除申请 | DELETE | /visit-apply/{id} | 删除申请 |

### 审批接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 通过申请 | POST | /approval/{id}/approve | 通过申请 |
| 拒绝申请 | POST | /approval/{id}/reject | 拒绝申请 |
| 获取审批列表 | GET | /approval/list | 获取审批列表 |

### 来访记录接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取记录列表 | GET | /visit-record/list | 获取来访记录列表 |
| 获取记录详情 | GET | /visit-record/{id} | 获取记录详情 |
| 签到 | POST | /visit-record/{id}/check-in | 访客签到 |
| 签退 | POST | /visit-record/{id}/check-out | 访客签退 |

### 统计接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 获取统计数据 | GET | /statistics | 获取统计数据 |
| 获取每日统计 | GET | /statistics/daily | 获取每日统计 |
| 获取访客统计 | GET | /statistics/visitor | 获取访客统计 |

## 响应格式

### 成功响应

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 分页响应

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "data": [],
    "total": 100,
    "page": 1,
    "size": 10
  }
}
```

### 错误响应

```json
{
  "code": 500,
  "message": "error message",
  "data": null
}
```

## 状态码说明

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未认证 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |
